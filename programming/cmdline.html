Linux command line (Bash) cheat sheet

This is a overview over useful Bash commands and shortcuts I probably will forget.

Bash is a Linux shell, but is also part of the Git tools on Windows (Git Bash).

Apart from the first command these tricks should work here too.

Don't loose heart because there is so many shortcuts.
Use the button to filter the list and choose a couple that will improve your life.
Then come back regularly (set an alarm ;-) and discover new ways to improve your mastery of the dark screen.

Current level: Nerdy

Command line tricks

General

ctrl+alt+t Start a new terminal (Only defalult in Ubuntu and Mint)

ctrl+c Stop command and return to prompt (Alternatively try Esc, ctrl+d, :q! )

ctrl++ Enlarge terminal and text

ctrl+- Shrink terminal and text

Navigation in directory tree

cd Always go to home directory when no arguments is present

cd .. Go a level up in the directory tree

cd - Go to last directory

~ Represents path to home directory (Ex: cd ~/Documents)

<Tab> Autocomplete path names. Write a couple of characters and press <Tab>.
If nothing happens there is multiple options for Autocompletion.
Press <Tab> again to see possible options.
Press (shift+)alt+? to see possible options when you know they are there, but can't remember what they are.
Press (shift+)alt+* to use all possible options (ls D(shift+)alt+* gives ls Documents Downloads. Same as ls D*, but feels cooler ;-)
<Tab>-autocompletion also works on commands and variables

Reuse commands

history Lists used commands

history n List the n last commands

↑/↓ Navigate last commands

ctrl+p Show the last command in history. Same as up arrow, but keeping hands on keyboard

ctrl+n Show the next command in history. Same as down arrow

ctrl+o Run current command in history and bring the next command up. Repeat ctrl+o to re-run a command sequence

!! Reuse entire line (Ex: sudo !!)

!string Reuse line starting with string

!?string Reuse line containing string

!n Re-run the nth line in history

!-n Re-run the nth-last line in history

ctrl+r Search history (Repeat for next match of search string) Ctrl+g to stop search. Use ctrl+j to be able to edit command

alt+. Insert previous argument (Repeat for earlier arguments)

Navigate command line

ctrl+←/→ Jump word

ctrl+f Move right (→) (Keeping hands on keyboard)

ctrl+b Move left (←) (Keeping hands on keyboard)

alt+f Jump after current word

alt+b Jump to beginning of word

ctrl+a/e Jump to start/end of line

ctrl+xx Jump to beginning of line - and back

ctrl+shift+c Copy text selected with mouse

ctrl+shift+v Paste text selected with mouse

Delete and insert

ctrl+k/u Clear to end/start of line

ctrl+w Delete word backwards

alt+d Delete to end of word

ctrl+d Delete character under cursor (Keeping hands on keyboard)

ctrl+h Delete character before cursor ((Backspace) Keeping hands on keyboard)

ctrl+y Insert deletetion at cursor position (Not regular ctrl+z!)

Change case

alt+u Make uppercase from cursor til end of word

alt+l Make lowercase from cursor til end of word

alt+c Capitalize letter under cursor and jump to end of word

Other

clear Clear terminal

alt+t Swap word with previous word

ctrl+t Swap letter under cursor with previous letter and move one character right

ctrl+s Freeze flow if screen is scrolling. Relict form teleprinters. Only mentioned to note that ctrl+q will resume flow

Useful stuff

leave 1230 Set an alarm to 12:30

leave +30 Set an alarm 30 minutes

ps Show running processes and their process id (PID)

kill nnnn Kill the proces with process id (PID) nnnn


gnome-system-monitor Runs Gnome's system monitor

gnome-system-monitor & Runs Gnome's system monitor in the background
